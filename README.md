# 📝 To-Do App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

🔗 https://8arsh.github.io/todo-app-js/

## ✨ Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* Data stored using LocalStorage

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
